Labyrinth Field - Maze game, collect points.

About Project

A simple maze-based game where players control a ball to collect 20 point boxes scattered throughout the maze.
This project was developed individually as part of a college assignment, focusing on basic game mechanics and interactive gameplay.

My Role

Independently developed the game

Features

  • Interactive gameplay with player-controlled ball movement
  • Point collection system with score tracking
  • Camera system that follows the player’s movement
  • Real-time score display (UI)
  • Smooth player movement and controls
  • Win condition after collecting all points

Tech Stack

  • Unity
  • C#
