Recipe App

Cooking is easier with simple recipes.

About Project

An Android-based recipe application developed using Android Studio as part of a college project.
The app allows users to explore simple recipes that can be easily followed at home by integrating external recipe APIs.

My Role

Independently developed the application.

Features

  • Fetch and display recipe data from external API
  • Simple and user-friendly interface
  • Navigation to detailed recipe view
  • External link to access full recipe instructions
  • Scrollable list using RecyclerView

Tech Stack

  • Android Studio
  • Kotlin
  • REST API
