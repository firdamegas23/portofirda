Product Dashboard

A website for managing various product information.

About Project

A web-based dashboard system built using Laravel to manage and streamline product-related data efficiently. This system includes various features like product information, pricing, and shipping details to support product management processes within the company.
This project was developed as part of a company system, focusing on functionality, data management, and usability.

My Role

Full-Stack Developer

Features

  • Product data management (input product, pricing, and shipping data)
  • Automated product code generation system
  • Import and export data using Excel
  • Generate and download product price details in PDF format
  • Role-based access control and user management

Tech Stack

  • PHP
  • CSS
  • Javascript
  • Bootstrap
  • Laravel
  • Nginx
  • MySQL (PhpMyAdmin)
  • Laragon
